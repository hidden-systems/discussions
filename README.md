## About This Repository

This repository powers the discussion layer for [Hidden Systems](https://pages.hiddensystems.org) via [Giscus](https://giscus.app/) — an open-source tool that maps GitHub Discussions to pages on the site. When you leave a comment on a Hidden Systems page, it lives here as a GitHub Discussion. No separate account needed beyond a GitHub login.

If you're reading this on GitHub and haven't visited the site yet, start there.

## What Is Hidden Systems?

Most of the systems that shape daily life — how food reaches your table, how water moves through a city, how energy gets from source to socket — are largely invisible. Hidden Systems is a free, interactive website that makes them visible.

Each topic is presented as a navigable graph of connected concepts, letting you move from broad overview to granular mechanism at your own pace. We believe that understanding these systems is a precondition for caring about them, and caring is a precondition for change.
